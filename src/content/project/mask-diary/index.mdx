## Which face do you show the world?

Mask Diary is an in-development narrative game built in Unity. Its branching data model connects social masks, daily choices, authored narrative, and multiple endings.

The project brings together original interface art, character masks, environmental illustration, sound, Scriptable Objects, JSON narrative data, and C# game systems.
